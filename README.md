# 🌾 AgriEdge Or-Mange Ltd - Salesforce OMS

## 📌 Project Overview

This project focuses on developing a Salesforce-based Order Management System (OMS) to improve agricultural supply chain efficiency.

## 🚀 Features

* Automated Order Processing
* Real-time Inventory Tracking
* Customer Service Integration
* Data Security & User Management
* Process Automation using Process Builder
* Apex Classes & Triggers

## 🛠️ Technologies Used

* Salesforce CRM
* Apex Programming
* Process Builder
* Data Modeling
